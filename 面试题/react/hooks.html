  • 使用函数更新状态
    • useState 返回的第二个值是一个函数，可以用来更新状态。另外，你还可以传递一个函数给这个函数，以便在更新状态时执行一些逻辑。这对于需要根据前一个状态计算新状态的场景非常有用
const [count, setCount] = useState(0);
function increment() {
  setCount((prevCount) => prevCount + 1);
}
  • 使用对象作为状态
    • useState 可以使用任何类型的值作为状态，包括对象和数组。如果状态是对象，可以使用扩展运算符或 Object.assign 创建一个新的状态对象，以便只更新需要更新的属性
const [person, setPerson] = useState({
  name: "Alice",
  age: 25,
  email: "alice@example.com",
});

function updateEmail(newEmail) {
  setPerson((prevPerson) => ({
    ...prevPerson,
    email: newEmail,
  }));
}
  • 惰性初始化状态
  • 在某些情况下，需要惰性初始化状态，即只有在首次渲染时才计算初始值。这可以通过将状态初始值指定为一个函数来实现
  • 我们将 useState 的初始值指定为一个函数，该函数执行一些计算，并返回计算结果作为初始状态值。由于函数只在初次渲染时执行一次，因此可以避免在每次重新渲染时都重复执行昂贵的计算。
const [count, setCount] = useState(() => {
  const initialCount = someExpensiveComputation();
  return initialCount;
});
