# 说下 vue 和 react 的区别

vue 提供自定义指令

减少在虚拟 dom 中的逻辑处理。

react 采用 jsx 语法

jsx 就是 xml 结合 js 的编译更快，安全性更高，创建模板（组件）方便快捷。

路由：

vue 路由：提供了路由实例，可以对路由进行管理，也提供了当前路由对象的信息，提供了导航守卫。

react 路由：提供了相关的组件以及当前路由对象的信息。

react 需要 hoc 进行封装导航守卫，（可以封装全局导航守卫，组件内导航守卫）。

vue 计算属性

Vue 提供了完整的计算属性 api ,其内部依赖改变计算属性重新计算并缓存。

react 本身没有计算属性，（但是在 class 类组件中可以使用 get/set 进行设置，或者函数式组件使用 object.defineProperty 的 get/set 进行设置） 。

vue 提供了 watch 属性

watch 属性可以 vue 状态依赖进行监听。

react 本身没有监听属性。

Vue 提供了 model 可以对自定义组件进行绑定

Vue 局部组件在使用需要在引入的组件中 components 进行挂载。

react 局部组件在引入组件中直接使用。

生命周期

在生命周期方面，vue 和 react 都提供了生命周期钩子，但 vue 的生命周期划分明确，从创建到销毁都是分工明确的，

而 react 的生命周期是阶段性的，同一种生命周期可以存在不同的阶段
