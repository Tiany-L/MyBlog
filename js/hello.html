# Hello World!

Js 是一门运行在浏览器端的脚步语言。
类似于 css，需要在 html 页面中引入 javascript。

# script 标签

我们几乎可以使用 <script> 标签将 JavaScript 程序插入到 HTML 文档的任何位置。

<!DOCTYPE html>
<html>
  <body>
    <p>script 标签之前...</p>

    <script>
      alert("Hello, world!");
    </script>

    <p>...script 标签之后</p>
  </body>
</html>

Suggest

内部脚本可以在任意地方执行（即 script 标签可以放在任意地方）但建议将 js 代码放在结构下边；body 结束标签前面；

<!DOCTYPE html>
<html>
  <body></body>
  <script>
    alert("Hello, world!");
  </script>
</html>

script 有一个 type 的属性，属性值是‘text/javascript’表示当前文本是 js 内容

<script type="text/javascript"></script>

# 引入外部 js

首先，需要编写一个外部的 js 文件，纯 js 代码
其次，在 html 页面中使用 script 标签来引入外部的 js

<script src="js/index.js"></script>

script 标签可以引入多次，也就是说，在一个 html 中，可以有多个 script 标签。

<script src="js/index.js"></script>
<script src="js/demo.js"></script>

注意细节

如果设置了 src 特性，script 标签内容将会被忽略。
一个单独的 <script> 标签不能同时有 src 特性和内部包裹的代码。

<script src="js/index.js">
  alert('Hello World'); //此内容会被忽略，不会执行，因为设定了 src。
</script>

我们必须进行选择，要么使用外部的<script src="…">，要么使用正常包裹代码的<script>。
为了让上面的例子工作，我们可以将它分成两个 <script> 标签。

<script src="js/index.js"></script>
<script>
  alert("Hello World");
</script>
